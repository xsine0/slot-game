# Slot-Game
It's is a simple slot game, built in React.  
There is no local storage or form submit, if you refresh the page all inputs are reseted.

## Registration
You need to provide a username.  
This username will no be saved, if you refresh the page you are promted to enter a username again

<img width="263" height="181" alt="image" src="https://github.com/user-attachments/assets/6c351f83-ebf9-4c47-9529-02e2e1e6d1ab" />

## Game
After providing a username, you will start with 1000 coins, these coins are nothing but text.  
To start the game click on _Get Lucky_

<img width="484" height="442" alt="image" src="https://github.com/user-attachments/assets/98caa7e5-7df1-48db-9b31-585d4754eba8" />

Every turn cost you 150 coins, however if you win, you earn 100 coins.  
There is no limit in the negative or positve number range.  

## Winning the Game
To win the game, all three emojis should be the same
